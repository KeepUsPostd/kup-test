Locations

Where can people review your brand? Add every physical location. Creators see the closest one when they submit a review.

🌐 Available everywhere
Turn ON for online-only / digital brands without a physical store. Reviews show in every user's "Near Me" feed regardless of distance, and you don't need to add any addresses below. Turn OFF if you have one or more brick-and-mortar locations.
Geo-verified submissions
When ON, creators can only submit reviews when their phone is at one of your locations. Reviews not filmed at the location are auto-rejected with a friendly message. Keep OFF if you accept off-site reviews.

Add a location

The address is geocoded automatically. If a location can't be found by address, the listing still saves — but distance-sorted ordering needs valid coordinates.

Remove this location?

It will be hidden from the franchise locator immediately. Existing reviews that referenced it stay intact.

Paste locations in bulk

One location per line. Format: Name, Street, City, State, ZIP. Name is optional — leave blank and the city is used as the label. Each row is geocoded server-side on save. Bad addresses still save (just without coordinates) so you can fix them later inline.
